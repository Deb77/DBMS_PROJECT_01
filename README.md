# DBMS_PROJECT_01

## Project setup

```
yarn install (in root directory)
yarn install (in client directory)
```

### Compiles and hot-reloads for development

```
nodemon server.js (in root directory)
yarn serve (in client directory)
```

### Lints and fixes files

```
yarn lint
```
